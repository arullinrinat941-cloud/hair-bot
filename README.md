# Hair Color Change Bot

Telegram бот для смены цвета волос на фотографии с помощью нейросети.

## Возможности
- Принимает от пользователя фотографию.
- Предлагает на выбор 8 цветов волос.
- Обрабатывает фото и возвращает результат с измененным цветом волос.
- Дает ссылку на нейросеть Qwen для улучшения качества.

## Используемые технологии
- Python 3.10
- Библиотеки: pyTelegramBotAPI, PyTorch, OpenCV, Pillow
- Хостинг: Bothost.ru
